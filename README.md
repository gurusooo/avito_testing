# Задание 1
## Высокий приоритет
* Поле “Команда” показывает текст “Вакансий пока нет”, хотя вакансии ниже отображаются, что вводит пользователя в заблуждение;
* Раздел “Ничего не нашлось” появляется между списками вакансий - это будет сбивать пользователя с толку;
* Фильтрация не учитывается: несмотря на выбранное предпочтение удалённой работы, выдаётся работа в офисе.

## Средний приоритет
* Кнопка "для людей с ОВЗ" недостаточно хорошо видна для людей с ОВЗ и сливается с другими;
* Бейдж “Аналитик” активен, но выдача включает руководителей и нерелевантные роли: например, несмотря на то, что data science в теге не был выбран, он всё равно отображается;
* Разные форматы указания локаций, нет единого стандарта;
* Отсутствие в вакансии "аналитик данных в Авито Авто" города.

## Низкий приоритет
* Неконсистентность: слова с буквой "ё" пишутся и через "е" и через "ё";
* Аватар пользователя не прогрузился;
* Кнопка “Сбросить всё” визуально выглядит равнозначной другим тегам - можно случайно нажать, в мобильной версии при скролле риск особенно велик;
* В вакансии "финансовый аналитик" текст "офис" находится левее, чем следует, также отсутствует тег снизу;
* Чекбокс "руководящая роль" неоправданно акцентирован.
